TVM

TVM is a Tiny Virtual Machine written in Javascript that includes an assembler and a virtual display.

Here is a link to a hosted copy of TVM with a sample program written in TVM assembler code that draws a rotating 3D cube http://www.physmo.com/tvmcube

Machine Features

The virtual machine has 8 32 bit registers, a stack, most common assembly instructions, labels and jumps, data declarations, and a 2 mode display device (text and graphics mode).
